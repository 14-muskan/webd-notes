CSS NOTES - styled using ELEMENT selector

Ways to add CSS in HTML
  • Inline CSS - using style as an attribute for an HTML tag, not considered as a good practice.
  • Internal CSS - using style as tag nested under head tag, useful for small webpages
  • External CSS - using a .css file and linking it with .html, considered as most important

Now another category of selectors - pseudo selectors

  • a:hover - to style the hover state
  • a:visited - to style the visited link
  • a:active - to style the active link
  • a:link - to style the link
  • p:link - to style the p tag wherever it is the first child of any parent tag

CSS- Box Model

I am box. With padding = 20 px which is applied within the element and margin = 30px which is applied outside of the element. The border is applied on the element which includes the padding but not the margin. So the shaded blue area has the div along with the padding until the border. After that, margin starts.
I am another box. Height of the box is 200px. But the total height covered by this div will be 200px + 2*margin + 2*border + 2*padding, similarly for width. But if we also add box-sizzing property as bprder box, then it will keep the total width excluding the margin as 200px.

Here the concept of margin collapse is also working which states that if the margin of two elements are overlapping, then the largest of them would be applied only.
