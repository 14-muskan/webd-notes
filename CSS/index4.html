FLEX BOX

Container Properties

  • The axis in which the items are laid flex-direction: column-reverse (here they will in the column direction in reverse order)is the main axis and the perpendicular axis is the cross axis
  • justify-content:center; centers the items along the main axis and align-items:center; centers along the cross axis.
  • When the cross axis starts, it is called cross-start, where it ends is called cross-end
  • flex-wrap: wrap; will wrap the content in the flex-direction
  • align-content: center; used when multiple lines are present. But there is no property like justify-items
  • using flex-flow: row wrap; we can simultaneously set the flex-direction and flex-wrap
  • gap: 10px; will make a gap of defined value between all the items.
  • row-gap: 2px; and column-gap: 2px; will set the gap property individually. We also have a shorthand for this gap: 2px 2px; first for the row then column

    • Item Properties

  • order is applied to each of the item, the element with least order will come first .order-1{order:1;} and .order-2{order:2;} Here first object will come first. Default order is 0
  • flex-grow:2; this will take up twice as spcae by other elements. Used on individual item
  • flex-shrink:2; will decide the ability of the item to shrink if needed, it will shrink to half the size
  • align-self: flex-start; this will decide the configuration of the particular element.
