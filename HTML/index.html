Core web vitals

CLS (CUMULATIVE LAYOUT SHIFT)
measure of how much does the layout shift when the page loads, if we set the height or width of the imgs in html, it will not shift, hence desirable

LCP (LARGEST CONTENTFUL PAINT)
Gives the time of the render of the largest element in the layout, ideally should be less than equal to 2.5 seconds

FID (FIRST INPUT DELAY)
The user input should start processing within 100 miliseconds

Generate a lighthouse report to analyze the performance of the webpage.

Ids are unique, cannot be assigned to multiple elements, one element cannot have multiple ids, helps in linking, tageted in css using #

Classes are not unique, can be assigned to multiple elements, one element can have multiple classes, targetted in css using .
